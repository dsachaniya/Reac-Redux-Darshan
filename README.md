# React - Redux Darshan

React - Redux Darshan which contains `react-router-dom`, `redux`, `jest`, `enzyme`, `eslint`, `husky` configured init.

## Clone the Repo

`Git clone https://github.com/dsachaniya/Reac-Redux-Darshan.git`

## Run Project

`npm start`

OR

`yarn start`

## Build Project

`npm run build`

OR

`yarn build`

## Serve Build

`npm i -g serve`

`npm run serve`

OR

`yarn serve`

## Test Project

`npm run test`

OR

`yarn test`
